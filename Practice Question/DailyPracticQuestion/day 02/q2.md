# C Program: Print Even Numbers till 20

## Problem
Write a C program to print all even numbers from 1 to 20.

---

## Algorithm Explanation

1. **Start**
2. Initialize a loop from 1 to 20.
3. For each number:
   - Check if the number is divisible by 2 (`num % 2 == 0`).
   - If true, print the number (itâ€™s even).
4. Continue until 20.
5. **End**

---

## C Program

```c
#include <stdio.h>

int main() {
    int i;

    // Step 1: Loop from 1 to 20
    printf("Even numbers from 1 to 20:\n");
    for(i = 1; i <= 20; i++) {
        // Step 2: Check if number is even
        if(i % 2 == 0) {
            printf("%d\n", i);
        }
    }

    return 0;
}
```
