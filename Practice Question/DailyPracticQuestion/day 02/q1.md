# C Program: Multiplication Table of a Number

## Problem
Write a C program to get a number from the user and print its multiplication table.

---

## Algorithm Explanation

1. **Start**
2. **Input a number** from the user.
3. Use a **loop** (from 1 to 10) to:
   - Multiply the given number with the loop counter.
   - Print the result in the format: `number x counter = result`.
4. **End**

---

## C Program

```c
#include <stdio.h>

int main() {
    int num, i;

    // Step 1: Ask user for input
    printf("Enter a number: ");
    scanf("%d", &num);

    // Step 2: Print multiplication table using a loop
    printf("Multiplication Table of %d:\n", num);
    for(i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", num, i, num * i);
    }

    return 0;
}
```
