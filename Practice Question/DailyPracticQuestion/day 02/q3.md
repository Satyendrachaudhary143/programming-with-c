# C Program: Print Odd Numbers till 20

## Problem
Write a C program to print all odd numbers from 1 to 20.

---

## Algorithm Explanation

1. **Start**
2. Initialize a loop from 1 to 20.
3. For each number:
   - Check if the number is **not divisible by 2** (`num % 2 != 0`).
   - If true, print the number (itâ€™s odd).
4. Continue until 20.
5. **End**

---

## C Program

```c
#include <stdio.h>

int main() {
    int i;

    // Step 1: Loop from 1 to 20
    printf("Odd numbers from 1 to 20:\n");
    for(i = 1; i <= 20; i++) {
        // Step 2: Check if number is odd
        if(i % 2 != 0) {
            printf("%d\n", i);
        }
    }

    return 0;
}
```
