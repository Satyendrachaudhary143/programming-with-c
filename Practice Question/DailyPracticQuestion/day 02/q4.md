# C Program: Check Whether a Number is Even or Odd

## Problem
Write a C program to check whether a given number is **even** or **odd**.

---

## Algorithm Explanation

1. **Start**
2. Take a number as input from the user.
3. Check if the number is divisible by 2:
   - If `num % 2 == 0`, then it is an **even** number.
   - Otherwise, it is an **odd** number.
4. Print the result.
5. **End**

---

## C Program

```c
#include <stdio.h>

int main() {
    int num;

    // Step 1: Ask user for input
    printf("Enter a number: ");
    scanf("%d", &num);

    // Step 2: Check even or odd
    if(num % 2 == 0) {
        printf("%d is an Even number.\n", num);
    } else {
        printf("%d is an Odd number.\n", num);
    }

    return 0;
}
```
