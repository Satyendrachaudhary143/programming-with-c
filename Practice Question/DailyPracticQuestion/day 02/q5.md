# C Program: Calculate Cube of a Number

## Problem
Write a C program to get a number from the user and calculate its cube.

---

## Algorithm Explanation

1. **Start**
2. Take a number as input from the user.
3. Calculate cube using the formula:  
   \[
   \text{cube} = \text{number} \times \text{number} \times \text{number}
   \]
4. Print the cube.
5. **End**

---

## C Program

```c
#include <stdio.h>

int main() {
    int num, cube;

    // Step 1: Get number from user
    printf("Enter a number: ");
    scanf("%d", &num);

    // Step 2: Calculate cube
    cube = num * num * num;

    // Step 3: Display result
    printf("The cube of %d is %d.\n", num, cube);

    return 0;
}
```
